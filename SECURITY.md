# Security Policy

## Supported Versions

Currently, we only support the latest version of `contextpack`.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within `contextpack`, please do **not** open a public issue. Instead, report it privately to the maintainers.

### Process
1. Email your findings to `130441797+theguega@users.noreply.github.com`.
2. Include a detailed description of the vulnerability, potential impact, and steps to reproduce.
3. We will acknowledge receipt of your report within 48 hours and provide a timeline for a fix if applicable.

We ask that you follow responsible disclosure guidelines and allow us time to address the issue before making it public.
